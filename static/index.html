Features

  • Brain Data Query: Retrieve goals, learnings, procedures, metrics, tasks, and self‑model.

Quick Start

1. Register for a free API key:

curl -X POST https://aidan-revenue-api.fly.dev/api/register \
  -H "Content-Type: application/json" \
  -d '{"email":"your@email.com","tier":"free"}'

2. Use the returned API key in the X-API-Key header:

curl -H "X-API-Key: sk_..." https://aidan-revenue-api.fly.dev/brain/status

3. Explore the interactive API documentation.
